ROCm RX 6700 XT Guide

Overview Installation Common Issues Resources
In this guide, I will show how I set up ROCm on Ubuntu 24.04 with an RX6700XT. As the AI era grows, having proper GPU support is essential. ROCm offers an open-source platform to fully leverage AMD GPUs.
Installation Steps (Ubuntu 24.04 Example)
Step 1 - Update baseline and install essentials 
sudo apt update
sudo apt upgrade
sudo apt install synaptic python3 python3-venv python3-pip git
Step 2 - Install ROCm and HIP libraries 
export HSA_OVERRIDE_GFX_VERSION=10.3.0
sudo apt install libamd-comgr2 libhsa-runtime64-1 librccl1 librocalution0 librocblas0 librocfft0 librocm-smi64-1 \
librocsolver0 librocsparse0 rocm-device-libs-17 rocm-smi rocminfo hipcc libhiprand1 libhiprtc-builtins5 radeontop
sudo usermod -aG render,video $USER
sudo reboot
After reboot, confirm ROCm is installed by running rocminfo. The first line should show "ROCk module loaded." Radeontop is optional for GPU monitoring.
